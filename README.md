# swxsoc-presentation

This repository contains a Reveal.js presentation intended to be hosted on GitHub Pages.

## Local preview

Open `/home/runner/work/swxsoc-presentation/swxsoc-presentation/index.html` in a browser.

## GitHub Pages hosting

1. Push changes to the repository default branch.
2. In GitHub, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Select the default branch and `/ (root)` folder.
5. Save and open the generated `https://<owner>.github.io/swxsoc-presentation/` URL.
